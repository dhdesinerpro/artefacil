Como usar esta arte

  1. Preencha os campos principais com os dados do seu negócio.
  2. Clique em “Gerar Prompt” e revise as informações.
  3. Confirme para gerar o prompt personalizado e baixe a imagem de referência.
  4. Abra o ChatGPT ou Gemini e cole o prompt junto com a imagem.
  5. Peça para manter o estilo da referência, trocando apenas os dados informados.
