# Arte Fácil Pro Delivery

Biblioteca de artes prontas com prompts personalizados para pequenos negócios, deliverys e empresas locais.

## Como funciona

1. Escolha uma arte no catálogo.
2. Preencha os dados do seu negócio.
3. Gere o prompt automaticamente.
4. Copie o prompt e baixe a imagem de referência.
5. Use no ChatGPT ou Gemini para criar sua arte final.

## Categorias atuais

- Hamburgueria
- Pastelaria
- Em expansão para pizzaria, açaí, marmitaria e mais.
